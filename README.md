## 简介
CML.ToolKitEx是一个基于C#的编程工具包，其中包含数据库操作、网络操作、配置操作等功能。

## 开发环境
* Language: C# 7.1
* IDE: Visual Studio 2019
* Framework: .Net Framework 4.0
